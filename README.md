# Таблицы Word в Excel

Простая программа для Windows, которая берет таблицы из файлов Word `.docx` и складывает их в один файл Excel `.xlsx`.

## Что умеет программа

- Обрабатывает все файлы `.docx` в выбранной папке.
- Достает все таблицы из каждого документа Word.
- Сохраняет каждую таблицу на отдельный лист Excel.
- Убирает лишние пробелы в ячейках.
- Пропускает полностью пустые строки.
- Показывает ход работы в окне программы.
- Пропускает поврежденные файлы и пишет об этом в журнале.

## Что нужно установить

Нужен установленный Python для Windows.

Если Python еще не установлен:

1. Откройте сайт <https://www.python.org/downloads/>.
2. Скачайте Python для Windows.
3. При установке обязательно поставьте галочку `Add Python to PATH`.
4. Завершите установку.

## Первый запуск

Перед первым запуском нужно один раз установить библиотеки:

1. Откройте папку с программой.
2. Дважды нажмите файл `install_requirements.bat`.
3. Дождитесь окончания установки.
4. Закройте окно, если оно не закрылось само.

## Как пользоваться

1. Дважды нажмите файл `run_docx_tables_app.bat`.
2. В поле `Папка с .docx` нажмите `Выбрать`.
3. Выберите папку, где лежат документы Word.
4. В поле `Excel-файл` нажмите `Сохранить как`.
5. Выберите, куда сохранить итоговый файл Excel.
6. Нажмите `Начать конвертацию`.
7. Дождитесь сообщения `Готово`.

## Важные заметки

- Программа работает только с файлами `.docx`.
- Старые файлы Word `.doc` не поддерживаются.
- Если итоговый Excel-файл уже открыт, закройте его перед запуском программы.
- Если в папке нет `.docx`, программа покажет ошибку.
- Если в документе нет таблиц, программа напишет об этом в журнале.

## Какие файлы здесь есть

- `docx_tables_to_excel_app.py` - сама программа.
- `run_docx_tables_app.bat` - запуск программы на Windows.
- `install_requirements.bat` - установка нужных библиотек.
- `requirements.txt` - список библиотек Python.
