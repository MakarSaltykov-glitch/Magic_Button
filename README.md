> Быстрый и простой конвертер таблиц между Word и Excel для Windows и macOS

## 🎯 Что это?

**Magic Button** - это приложение для переноса таблиц между документами Word (.docx) и электронными таблицами Excel (.xlsx) одним нажатием кнопки.

✅ **Работает на Windows и macOS**  
✅ **Не требует установки Python** (готовые приложения)  
✅ **Простой интерфейс** - просто нажимай кнопки  
✅ **Бесплатное и открытое** - MIT лицензия  

Идеально для:
- 📋 Быстрого преобразования отчетов и данных
- 🔄 Передачи таблиц между системами
- ⚡ Массовой обработки нескольких файлов
- 💼 Подготовки документов для отправки

## ✨ Возможности

### Mode 1: Word → Excel 📄
- Берет **ВСЕ** файлы `.docx` из выбранной папки
- Извлекает все таблицы из каждого файла
- Каждую таблицу сохраняет на **отдельный лист** в Excel
- Автоматически очищает пустые строки
- Убирает лишние пробелы в ячейках
- Обрабатывает поврежденные файлы без сбоев

### Mode 2: Excel → Word 📊
- Берет выбранный `.xlsx` файл
- Все непустые листы преобразует в Word-таблицы
- Каждый лист становится отдельной таблицей с заголовком
- Автоматически удаляет пустые столбцы
- Аккуратно форматирует текст

## 📖 Как использовать

### Word → Excel

```
1. Откройте программу
2. Перейдите на вкладку "📄 Word → Excel"
3. Нажмите "Выбрать" и укажите папку с файлами .docx
4. Нажмите "Сохранить как" и выберите имя для файла .xlsx
5. Нажмите "Начать: Word → Excel"
6. Дождитесь сообщения "Готово"
```

**Результат:** В Excel-файле будет столько листов, сколько таблиц найдено.

### Excel → Word

```
1. Откройте программу
2. Перейдите на вкладку "📊 Excel → Word"
3. Нажмите "Выбрать" и укажите .xlsx файл
4. Нажмите "Сохранить как" и выберите имя для файла .docx
5. Нажмите "Начать: Excel → Word"
6. Дождитесь сообщения "Готово"
```

**Результат:** Word-файл с таблицами из всех листов Excel.

## 🔗 Зависимости

Программа использует только стандартные библиотеки:
- **python-docx** - работа с Word документами
- **pandas** - работа с таблицами данных
- **openpyxl** - работа с Excel файлами
- **tkinter** - графический интерфейс (встроен в Python)

## ⚠️ Важно знать

- ✅ Поддерживаются только `.docx` и `.xlsx` (новые форматы)
- ❌ Старые форматы `.doc` и `.xls` **не поддерживаются**
- ✅ Работает с таблицами любого размера
- ✅ Автоматически пропускает пустые таблицы и листы
- ✅ Обрабатывает поврежденные файлы без краша
- 📝 Весь процесс логируется в окне программы

## 🐛 Если что-то не работает

**Проблема:** "tkinter not found"
```
Решение: При установке Python поставьте галочку "tcl/tk and IDLE"
```

**Проблема:** "No module named 'docx'"
```
Решение: Запустите install_requirements.bat ещё раз
```

**Проблема:** "Access denied" при сохранении
```
Решение: Закройте файл в Excel/Word перед конвертацией
```

1. Откройте вкладку `Excel в Word`.
2. В поле `Excel-файл .xlsx` выберите Excel-файл.

Программа возьмет все непустые листы Excel и создаст Word-документ с таблицами.

Это необязательно. Программа уже запускается через `run_docx_tables_app.bat`.

Если нужен один файл приложения:

1. Дважды нажмите `build_exe.bat`.
2. Дождитесь окончания сборки.
3. Готовый файл появится в папке `dist`.
4. Запускайте файл `MagicButton.exe`.

## Если что-то пошло не так

- Если программа не запускается, проверьте, установлен ли Python.
- Если окно быстро закрывается, сначала запустите `install_requirements.bat`.
- Если итоговый файл уже открыт в Word или Excel, закройте его и повторите запуск.
- Если файл поврежден, программа пропустит его и напишет сообщение в журнал.
- Если нужен старый формат `.doc` или `.xls`, сначала откройте файл в Word или Excel и сохраните его как `.docx` или `.xlsx`.
