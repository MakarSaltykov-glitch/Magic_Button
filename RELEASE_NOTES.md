# Magic Button v2.1 - Release Notes

## 🎉 Основные возможности

**Magic Button** - это простое приложение для конвертации таблиц между **Word** и **Excel** без установки Python!

### ✨ Новое в v2.1:

- ✅ **macOS поддержка**: Готовый `.app` для Mac (Intel и Apple Silicon)
- ✅ **Windows поддержка**: Готовый `.exe` для Windows 7/8/10/11  
- ✅ **Ноль зависимостей**: Просто скачайте и запустите!
- ✅ **Двусторонняя конвертация**:
  - Word → Excel (все таблицы в один файл)
  - Excel → Word (каждый лист становится таблицей)

---

## 📥 Установка

### 🍎 macOS

**Способ 1: DMG Installer (рекомендуется)**
```bash
1. Скачайте MagicButton-installer.dmg
2. Откройте DMG файл (двойной клик)
3. Перетащите MagicButton в папку Applications
4. Откройте из Launchpad или Finder
```

**Способ 2: Прямой запуск APP**
```bash
1. Скачайте MagicButton.app
2. Откройте: right-click → Open (первый раз)
3. Или: open MagicButton.app
```

### 🪟 Windows

```bash
1. Скачайте MagicButton.exe
2. Двойной клик на файл
3. Приложение запустится в отдельном окне
```

> **Первый запуск может занять несколько секунд** - это нормально, программа распаковывает свои файлы.

## 🚀 Использование

### Word → Excel

```
1. Откройте приложение MagicButton
2. Перейдите на вкладку "Word → Excel"
3. Выберите папку с файлами .docx
4. Нажмите "Конвертировать"
```

**Результат**: Все таблицы со всех Word документов объединяются в один Excel файл (каждый документ = отдельный лист)

### Excel → Word

```
1. Откройте приложение MagicButton
2. Перейдите на вкладку "Excel → Word"
3. Выберите файл .xlsx
4. Нажмите "Конвертировать"
```

**Результат**: Каждый лист Excel становится таблицей в новом Word документе

## 🐛 Известные проблемы

### macOS: "Magic Button не может быть открыто..."
Решение:
```bash
xattr -d com.apple.quarantine ~/Downloads/MagicButton.app
```

### Windows: SmartScreen предупреждение
- Это нормально для неподписанных приложений
- Нажмите "Подробнее" → "Всё равно запустить"
- Или подпишите EXE кодом (для корпоративных окружений)

### Файлы "съедаются" в конвертации
- Копируйте исходные файлы перед конвертацией
- Приложение НЕ удаляет исходные файлы
