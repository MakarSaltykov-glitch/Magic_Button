## Установка

### ⚡ Самый простой способ (БЕЗ Python)

**Windows:**
1. [Скачайте MagicButton.exe](https://github.com/MakarSaltykov-glitch/Magic_Button/releases)
2. Дважды нажмите на файл
3. Готово! ✅

**macOS:**
1. [Скачайте MagicButton.app.zip](https://github.com/MakarSaltykov-glitch/Magic_Button/releases)
2. Разархивируйте файл
3. Перетащите MagicButton.app в Applications
4. Готово! ✅

### Способ 2: Установка с Python (для разработчиков)

#### Шаг 1: Установка Python

Если Python не установлен:
1. Перейдите на https://www.python.org/downloads/
2. Скачайте Python 3.7 или выше
3. **ВАЖНО:** При установке поставьте галочку **"Add Python to PATH"**
4. Завершите установку

#### Шаг 2: Установка зависимостей

**Windows:**
1. Откройте папку с программой
2. Дважды нажмите на файл `install_requirements.bat`
3. Дождитесь завершения установки
4. Закройте окно

**macOS/Linux:**
```bash
pip3 install -r requirements.txt
```

## Режимы работы

### Режим 1: Word → Excel 📄

**Назначение:** Извлечение таблиц из Word-документов в Excel

**Как работает:**
1. Выбираете папку с `.docx` файлами
2. Программа ищет все `.docx` файлы в этой папке
3. Извлекает из них все таблицы
4. Каждую таблицу сохраняет на отдельный лист в Excel

**Инструкция:**
```
1. Откройте вкладку "📄 Word → Excel"
2. Нажмите "Выбрать" рядом с "Папка с Word-файлами .docx"
3. Выберите папку, где лежат ваши .docx файлы
4. Нажмите "Сохранить как" рядом с "Куда сохранить Excel .xlsx"
5. Укажите, где и как сохранить результирующий .xlsx файл
6. Нажмите кнопку "Начать: Word → Excel"
7. Дождитесь сообщения "Готово" в журнале
8. Excel-файл готов к использованию
```

**Пример:**
- Входные файлы:
  - `/Documents/reports/report1.docx` (содержит 3 таблицы)
  - `/Documents/reports/report2.docx` (содержит 1 таблицу)

- Результат: `tables.xlsx` с 4 листами
  - Лист 1: таблица из report1.docx
  - Лист 2: таблица из report1.docx
  - Лист 3: таблица из report1.docx
  - Лист 4: таблица из report2.docx

### Режим 2: Excel → Word 📊

**Назначение:** Преобразование листов Excel в Word-таблицы

**Как работает:**
1. Выбираете один `.xlsx` файл
2. Программа читает все непустые листы
3. Каждый лист преобразует в таблицу в Word
4. Сохраняет всё в один `.docx` документ

**Инструкция:**
```
1. Откройте вкладку "📊 Excel → Word"
2. Нажмите "Выбрать" рядом с "Excel-файл .xlsx"
3. Выберите один .xlsx файл
4. Нажмите "Сохранить как" рядом с "Куда сохранить Word .docx"
5. Укажите, где и как сохранить результирующий .docx файл
6. Нажмите кнопку "Начать: Excel → Word"
7. Дождитесь сообщения "Готово" в журнале
8. Word-файл готов к использованию
```

**Пример:**
- Входной файл: `data.xlsx` (содержит 3 листа)
  - Лист "Sales"
  - Лист "Expenses"
  - Лист "Summary"

- Результат: `report.docx` с 3 таблицами
  - Таблица "Sales"
  - Таблица "Expenses"
  - Таблица "Summary"

## Примеры

### Пример 1: Быстрое преобразование отчета

**Задача:** У вас есть Word-документ с таблицами, нужно отправить их в Excel

**Решение:**
```
1. Дважды нажмите run_docx_tables_app.bat
2. Убедитесь, что выбрана вкладка "📄 Word → Excel"
3. Нажмите "Выбрать" и укажите папку с документом
4. Нажмите "Сохранить как" и назовите файл tables.xlsx
5. Нажмите "Начать: Word → Excel"
6. Готово! Откройте tables.xlsx
```

**Время:** ~10 секунд

### Пример 2: Массовая обработка

**Задача:** Нужно обработать 50 Word-документов с таблицами

**Решение:**
```
1. Поместите все 50 файлов в одну папку
2. Запустите программу
3. Выберите эту папку
4. Нажмите "Начать"
5. Программа обработает все 50 файлов автоматически
6. В Excel-файле будут все таблицы со всех 50 документов
```

**Результат:** Все таблицы в одном Excel-файле, каждая на отдельном листе

## Решение проблем

### ❌ Ошибка: "tkinter not found"

**Причина:** Python установлен без компонента tkinter

**Решение:**
- Переустановите Python с галочкой "tcl/tk and IDLE"
- Или установите tkinter отдельно:
  ```
  python -m pip install tk
  ```

### ❌ Ошибка: "No module named 'docx'"

**Причина:** Не установлены зависимости

**Решение:**
```
1. Дважды нажмите install_requirements.bat
2. Дождитесь завершения
3. Запустите программу снова
```

### ❌ Программа не запускается

**Решение:**
1. Откройте Command Prompt (Win+R → cmd)
2. Перейдите в папку с программой:
   ```
   cd C:\path\to\Magic_Button
   ```
3. Запустите программу напрямую:
   ```
   python docx_tables_to_excel_app.py
   ```
4. Смотрите ошибку в консоли

### ❌ "Access denied" при сохранении

**Причина:** Файл открыт в другой программе

**Решение:**
- Закройте файл в Excel/Word
- Запустите конвертацию снова

### ❌ Результирующий файл пустой

**Причина:** В исходных файлах нет таблиц

**Решение:**
- Проверьте, что в исходных файлах есть таблицы
- Смотрите журнал обработки в программе
- Попробуйте выбрать правильную папку/файл

### ❌ Таблицы некорректно преобразованы

**Решение:**
1. Проверьте формат исходного файла (.docx или .xlsx)
2. Убедитесь, что в таблицах нет неправильных символов
3. Попробуйте сохранить исходный файл заново
4. Запустите конвертацию снова

## Советы и хитрости

✅ **Совет 1:** Для быстрого преобразования можно перейти в папку с программой и запустить .bat файл из контекстного меню

✅ **Совет 2:** Если много файлов, лучше поместить их все в одну папку и обработать за один раз

✅ **Совет 3:** Проверьте журнал обработки после завершения - там видны все ошибки

✅ **Совет 4:** Пустые таблицы и пустые листы автоматически пропускаются

✅ **Совет 5:** Программа не изменяет исходные файлы - только создает новые
