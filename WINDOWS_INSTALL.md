# 🪟 Magic Button для Windows - Установка

## Для обычного пользователя (не требуется знание ПК!)

### Шаг 1️⃣: Скачайте установщик
```
Скачайте файл: MagicButton-Installer.exe
```

### Шаг 2️⃣: Установите
```
1. Дважды клик на MagicButton-Installer.exe
2. Нажмите "Next" → "Next" → "Install" 
3. Подождите пару секунд
4. Готово! ✅
```

### Шаг 3️⃣: Используйте
```
На рабочем столе или в Пуск → Magic Button → кликайте!
```

---

## Что делает Magic Button?

✅ **Конвертирует Word файлы в Excel**
   - Выберите папку с файлами .docx
   - Нажмите кнопку
   - Получите .xlsx со всеми таблицами

✅ **Конвертирует Excel в Word**
   - Выберите файл .xlsx
   - Нажмите кнопку
   - Получите .docx со всеми таблицами

## 🆘 Проблемы?

### "Windows SmartScreen предупреждение"
1. Нажмите "Подробнее"
2. Нажмите "Всё равно запустить"
3. Это нормально - приложение не подписано, но безопасно

### "Файлы не конвертируются"
1. Убедитесь что файлы .docx или .xlsx в папке
2. Попробуйте скопировать их в простую папку (C:\temp\docs)
3. Выберите эту папку

### "Удалить программу"
1. Пуск → Параметры → Приложения → Приложения и возможности
2. Найдите "Magic Button"
3. Нажмите "Удалить"
