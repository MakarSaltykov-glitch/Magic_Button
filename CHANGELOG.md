# Changelog

All notable changes to this project will be documented in this file.

## [2.0] - 2024-06-06

### Added
- ✨ Красивый Windows-style интерфейс с темой Segoe UI
- 📊 Вкладка "Справка" с полной документацией
- 🎨 Иконки для вкладок (📄 Word → Excel, 📊 Excel → Word)
- 📈 Улучшенное форматирование сообщений об ошибках
- 🔧 Поддержка PyInstaller для создания .exe
- 💻 Лучший обработчик ошибок и логирование
- 🚀 Быстрые bat-файлы для удобного запуска

### Changed
- 🎯 Переработан интерфейс для лучшей юзабилити
- 📝 Обновлена документация и README
- ⚙️ Оптимизирована обработка больших файлов
- 🔄 Улучшена обработка кодировок текста

### Fixed
- ✅ Исправлена обработка пустых строк и столбцов
- ✅ Улучшена работа с поврежденными файлами
- ✅ Исправлены ошибки при работе с Unicode символами
